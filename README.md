# Shell Spar Express Ottawa

A modern, responsive website for Shell Spar Express Ottawa - a 24/7 fuel station, car wash, and SPAR Express convenience store located in Verulam, South Africa.

## 📋 Features

- 🚗 **Car Wash Booking** - Online booking system for car wash packages
- ⛽ **Fuel Information** - Premium Shell fuels (V-Power, FuelSave)
- 🛒 **SPAR Express** - Convenience store with fresh food and groceries
- 🏧 **ATM Services** - FNB, ABSA, and Cash Express ATMs
- 📦 **Courier Guy Locker** - Package pickup and drop-off
- 💧 **Water Refill** - R1 per litre drinking water
- 📱 **Mobile Responsive** - Works perfectly on all devices
- 💬 **WhatsApp Integration** - Floating button for instant customer contact

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Swiper.js (image gallery)
- AOS (scroll animations)
- Iconify (icons)
